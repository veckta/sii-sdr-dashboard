# Veckta SDR - Dashboard 📊

Panel de control y métricas desarrollado con **Streamlit** para la auditoría y monitoreo en tiempo real del sistema SDR de **SII**.

## 🚀 Características Principales
- **KPIs en Tiempo Real:** Visualización rápida de métricas clave (Conversaciones Únicas, Abonos Cerrados, Leads Alta Escala, Descartados).
- **Auditoría de Chats:** Interfaz para revisar el historial completo de conversaciones de cada lead.
- **Integración con Supabase:** Conexión directa vía REST API al historial de sesiones para leer la clasificación generada por el bot.

## 🛠️ Stack Tecnológico
- **Frontend / Framework:** Python, Streamlit
- **Datos:** Pandas
- **Conectividad:** HTTPX, Supabase REST API
